# 小琥哥官網 (Minimalist Premium Branding Site)

這是為「小琥哥」打造的官方品牌網站，旨在展現極簡奢華感 (Minimalist Premium) 並提升品牌形象。

## 技術棧 (Tech Stack)

- **框架**: [Next.js 15 (App Router)](https://nextjs.org/)
- **語言**: [TypeScript](https://www.typescriptlang.org/)
- **樣式**: [Tailwind CSS](https://tailwindcss.com/)
- **開發規範**: ESLint

## 資料夾結構 (Folder Structure)

- `src/app`: 路由與頁面組件 (App Router)
- `src/components`: 共享的 React 組件
- `src/lib`: 工具函數與配置
- `src/styles`: 全域樣式配置 (可根據需求擴展)
- `public`: 靜態資源（圖片、字體等）

## 開發指令

### 啟動開發伺服器
```bash
npm run dev
```

### 構建專案
```bash
npm run build
```

### 啟動構建後的伺服器
```bash
npm run start
```

## 設計風格指南

- **色調**: 極簡灰、奢華黑、純粹白。
- **字體**: 優雅的襯線體與乾淨的無襯線體。
- **動效**: 細微且流暢的過渡，避免過度花哨。
